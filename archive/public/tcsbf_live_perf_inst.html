Those cracks should be flowers - Live Performance Instructions

There are 7 invisible buttons:

Bottom-Left - Add a pitched light-pole

Bottom-Right - Add a beating oscillator

Top-Left - Remove a pitched light-pole

Top-Right - Remove a beating oscillator

Center-Right-Edge - Start Crescendo/Descrendos

Center-Left-Edge - Stop Crescendo/Descrendos

Center-Middle - Kill All Audio (cannot be undone)

Once the score is loaded, touch anywhere on the screen to activate buttons and start piece.
